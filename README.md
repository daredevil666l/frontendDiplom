# Дипломный проект "ИдёмВКино"

Привет! Это мой дипломный проект по курсу React.
Приложение для покупки билетов в кино. Есть админка для настройки залов и сеансов, и клиентская часть для выбора мест.

## Как запустить

1.  Установи зависимости: `npm install`
2.  Запусти проект: `npm run dev`

## Что сделано

*   **Админка:**
    *   Можно создавать и удалять залы.
    *   Настраивать конфигурацию залов (ряды, места, VIP-места).
    *   Устанавливать цены.
    *   Добавлять фильмы (с постерами!).
    *   Составлять расписание сеансов (перетаскивание фильмов на сетку).
    *   Открывать продажи билетов.
*   **Клиентская часть:**
    *   Просмотр расписания на сегодня и ближайшие дни.
    *   Выбор фильма и сеанса.
    *   Выбор мест в зале (обычные и VIP).
    *   Покупка билетов (генерация QR-кода).

## Технологии

*   React
*   Redux Toolkit (для хранения данных)
*   React Router (для навигации)
*   Axios (для запросов к серверу)
*   React DnD (для перетаскивания фильмов в админке)

Старался писать понятный код, разбил все на компоненты. Данные хранятся на сервере Нетологии.
